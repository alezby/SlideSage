# **App Name**: Slide Sage

## Core Features:

- Google Account Connection: Connect with the user's Google account to access Google Slides presentations.
- Presentation Selection: Allow users to select a presentation from their Google Drive.
- LLM-Powered Analysis: Use an LLM tool to analyze the content of each slide in the presentation based on a user-provided prompt (e.g., brand identity).
- Automated Commenting: Automatically add comments to the Google Slides presentation where the LLM tool suggests changes based on the prompt.
- Comment Summarization: Summarize the comments, slide-by-slide or overall.
- Slide Navigation: Provide an easy way to navigate through the slides.
- Conversational Agent: Implement a conversational agent based on Google Agent Development Kit for interactive prompt refinement and result review.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#F5F5F5), creating a clean and modern backdrop.
- Accent color: Teal (#009688), to highlight important comments.
- Body and headline font: 'Inter' (sans-serif) for a modern, objective look.
- Use simple and professional icons to represent different functionalities like 'upload', 'analyze', and 'comment'.
- A clean, organized layout to present presentation slides alongside the LLM analysis and comments.
- Smooth transitions between slides and sections for a fluid user experience.
